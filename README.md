# 🌸 Iris Flower Classification using K-Nearest Neighbors (KNN)

## 📌 Project Overview

This project implements a Machine Learning Classification Model using the famous Iris Dataset and the K-Nearest Neighbors (KNN) algorithm.

The model predicts the species of Iris flowers based on their petal and sepal measurements.
